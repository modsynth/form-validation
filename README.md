> React Hook Form validation module with Zod schemas

Part of the [Modsynth](https://github.com/modsynth) ecosystem.

## Features

- React Hook Form integration
- Zod schema validation
- TypeScript support
- ✨ **Common Validation Schemas**: Email, password, phone (v0.2.0)
- ✨ **Helper Functions**: Form validation utilities (v0.2.0)

## What's New in v0.2.0

- **Validation Schemas**: Pre-built Zod schemas (email, password, phone, etc.)
- **Form Utilities**: Helper functions for common validation tasks

## Installation

```bash
npm install @modsynth/form-validation react-hook-form zod
```
